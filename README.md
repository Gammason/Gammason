# Elijah

Experimental physicist working on electromigration, diffusion, and thin-film systems. My work focuses on combining experimental techniques with Python-based tools for data analysis, modeling, and instrument automation.

## Core Projects

### Electromigration Analysis Toolkit
Python GUI for analyzing electromigration experiments, including automated detection of onset current using derivative and plateau-based methods.

- Rmin and Rmax analysis  
- dR/dI-based onset detection  
- Multi-cycle comparison  
- Interactive visualization  

---

### Diffusion Analysis Toolkit
Physics-based modeling tool for diffusion processes using Arrhenius relations.

- Diffusion coefficient and length calculation  
- Multiple mechanisms (lattice, grain boundary, surface, effective)  
- Temperature and current-based analysis  
- Parameterized physical modeling  

---

### Electromigration Control System
Automated experimental platform for electromigration measurements using a Keithley SMU.

- Pulse-based electromigration protocol  
- Real-time measurement and visualization  
- Rmax and Rmin extraction  
- Resistance vs time measurement  
- Automated data logging  

---
